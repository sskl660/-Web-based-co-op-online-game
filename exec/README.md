# 🕹How to Run

### Sample URL

- 배포 서버 주소 : https://k5a302.p.ssafy.io/

**Frontend**

```sh
cd frontend
yarn install
yarn start
```

**Backend**

```sh
cd backend
./gradlew build
cd build/libs
java -jar PLAYSSAFY-0.0.1-SNAPSHOT.jar
```
